About kepler.cafe

kepler.cafe is a small independent development studio focused on creating elegant, useful applications for Apple platforms. Each project aims to solve real problems for users while maintaining high standards for privacy, performance, and design.

I build apps that I want to use myself, focusing on utility, thoughtful design, and user experience. Whether it's enhancing macOS window management with DockDoor, personalizing your dock with CharmBar, or providing a privacy-focused Reddit experience with OpenArtemis.
